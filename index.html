Bulls and Cows Game

This is a C++ isogram guessing game running in the Unreal Engine. Some of the challenges I found whilst making this game are making the game check to see if the string had any characters that were repeating, and making sure the word was the correct length. Another challenge I overcame was having the game load the isogram words from an external file. I enjoyed the challenges this project presented to me and the knowledge I gained from doing it. **I did not make any of the actors in this project, I only programmed the terminal interaction.

Number Game

This is a purely C++ number game that runs in the console. This was the first thing I made in C++ and it is a game where you get a sum and product of three numbers and you have to figure out what those three numbers are. The challenges that were presented to me in making this were how to properly output to console, how to display instructions that a user understands, and how to get a random value in C++.

Building Escape

This game is a C++ puzzle game made in Unreal Engine. The objective of the game is to escape with the goblet at the end, you must add enough weight to the pedestal to hold the door open at the end. Some challenges I met when making this game were basics of level design, lighting, how to access properties of objects from within code, and how to set up and use prefabs. **I did not make any of the actors in this project, I only programmed the gameplay and created the level.

Toon Tanks

This is a third-person tank game in which you drive around and destroy stationary turret enemies, which is made with C++ in Unreal Engine, you win the game when you destroy all of the turrets. Some of the challenges I met and overcame as well as some of the things I learned from completing the project are: Actor lifespans, how to destroy an actor, how to remove control upon game over (win or lose), how to make prefabs with inherited properties as well as individual properties, and how to manage player and enemy health. **I did not make any of the actors in this project, I only programmed the gameplay.
